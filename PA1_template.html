3.2. Fill in all of the missing values in the dataset. Create a new dataset that is equal to the original dataset but with the missing data filled in
To impute missing values, the mean across all days for the 5-minute interval was used to impute missing values(NA). The implementation was inspired by this stackoverflow thread

imputedData <- mydf %>%
               group_by(interval) %>%
               mutate(steps = ifelse(is.na(steps), mean(steps, na.rm = TRUE), steps))
